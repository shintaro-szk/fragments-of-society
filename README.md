# Fragments of Society

[Documentation](./docs/README.md)

Simple 2D top-down dungeon game about real social issues.

---

## Quick Info

- **Genre:** Top-down pixel dungeon crawler
- **Inspiration:** Soul Knight, Super Mario
- **Structure:** Level → Enemies → Mid Area → Boss → Note Reward

## Themes

1. **Pollution**
2. **Drugs**
3. **Corruption**
